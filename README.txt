SCHEDULING SYSTEM
A GUI-based scheduling system was developed to interface with a MySQL database for a global consulting organization.

IntelliJ IDEA 2021.3.3 (Community Edition)
Oracle OpenJDK version 17.0.2
JavaFX-SDK-19

Open the folder containing the project in IntelliJ and update paths for JavaFX, JDK and JDBC.
Run the program in IntelliJ or via the command line.
Login with username: admin password: admin

An additional report on the Reports form are past appointments from the current time.
